# React + Next.js Practice Project

مشروع صغير للتدرّب على React و Next.js قبل المقابلة:

- صفحة رئيسية تعرض قائمة مواضيع
- صفحة تفاصيل لكل موضوع
- API Route للتعامل مع البيانات
- SEO باستخدام `<Head>`
- استخدام Link و Image من Next.js

## تشغيل المشروع

1. نزل الكود:
```bash
git clone https://github.com/USERNAME/react-next-practice.git
cd react-next-practice
```

2. ثبّت التبعيات:
```bash
npm install
```

3. شغّل بيئة التطوير:
```bash
npm run dev
```

4. افتح المتصفح على `http://localhost:3000`
